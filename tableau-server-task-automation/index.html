The lazy Tableau Server admin's guide to automation

Being a Tableau Server admin can be boring. There I said it. Sometimes you’re stuck doing manual tasks or fixing problems that you already know the answer to……and that ain’t always engaging stuff, when all you really wanna do is watch some videos of cats (dear god, NO NOT THE MOVIE) or some cam footage of Russian drivers randomly crashing into each other for no apparent reason. Luckily, I’ve found the solution. This series is about how you too can spend more time watching internet memes, instead of administering Tableau Server - via the magic of automation. There are automation options for many different aspects of Tableau Server, such as installation, monitoring, and testing. However, this series will focus on common tasks such as creation of projects/groups, addition of users to sites/group, and deployment of workbooks/datasources.
